Previous Coding Experience

  The only previous experience with coding/computer science that I hads was in High School. My senior year I took AP Computer Science. It was a class based on the JavaScript coding language. The class was not completely impossible, but it was fairly difficult. Especially since it was my very first exposure to coding, I had a hard time understanding some of the concepts. The teacher was not very helpful and she was not very easy to approach to ask questions which is one of the reasons I did so poorly. I did learn a lot about coding and what it entails but I really wish I took the class more seriously and perhaps I would have done better in the end.

I believe that I had a badintroduction to the world of computer science.
